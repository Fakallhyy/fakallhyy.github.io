git使用

1.初始化

  • git init
    .git目录存放的是本地库相关的子目录和文件，不要删除，不要乱动

2.设置签名

区分不同开发人员的身，和账号 密码无关

项目级别/仓库级别:仅在当前本地库范围有效

  • git config user.name tom_pro
  • git config user.email goodMorning_pro@xxx.com
    信息保存位置:./.git/config

系统同户籍别：登录当前操作系统的用户范围

  • git config –global user.name tom_pro
  • git config –global user.email goodMorning_pro@xxx.com
    信息保存位置:~/.gitconfig

git status —– 暂存区状态查看

git add xx.xx –添加到暂存区

git commit -m “commit message” [file name] 将暂存区的内容提交到本地库

3.版本

git log 查看历史记录
git log –pretty=online
git reflog

回滚
git reset –hard 索引值
–hard 在本地库移动HEAD指针 重置暂存区，重置工作区

4.删除文件

5.分支

git branch -v 查看当前所有分支
git branch xxx 创建分支
git checkout xxx 切换分支

切换到master
git merge xxxx 合并xxx分支到master
